Life, the Universe, and Everything
----------------------------------

This package defines a viewlet with an editable image, could be used to replace the plone logo.

Don't Panic
-----------

Configure
^^^^^^^^^

- Go to 'Site Setup'.
- Select 'headband'.
- Upload an image.
